# 題目

Given two strings s and t which consist of only lowercase letters.
String t is generated by random shuffling string s and then add one more letter at a random position.
Find the letter that was added in t.

給兩個字串 s 和 t ，兩個都只包含小寫文字。
字串 t 是由 s 的文字隨機打亂組成，再加入一個隨機的文字在隨機的位置。
找出那個隨機的文字。

Example:

``` swift
Input:
s = "leetcode"
t = "lteweodce"

Output:
w
```


# 第一次解題 : 33 ms
算出 s 跟 t 的 bit 加總，相減後就可以得出多出來的 bit ，轉回 Character 就是答案。

``` swift
func findTheDifference(_ s: String, _ t: String) -> Character {
	var sumS = [UInt8](s.utf8).map { Int($0) }.reduce(0){$0 + $1}
	var sumT = [UInt8](t.utf8).map { Int($0) }.reduce(0){$0 + $1}
	return Character(UnicodeScalar(sumT - sumS)!)
}
```


# Other Solution
利用 XOR 相同會回傳 0 ，將字串組成後依序執行 XOR，最後會只剩下不同的那一個。

``` swift
func findTheDifference(_ s: String, _ t: String) -> Character {
    let xor = (s + t).utf8.reduce(0, { $0 ^ $1})
    return Character(UnicodeScalar(xor))
}
```
